本系統整合動態閾值心率偵測、斜率式呼吸偵測與膚電反應分析，提供即時壓力指數量化監測。

壓力指數公式
S_HR = clamp((BPM − Base_BPM×1.20) / (Base_BPM×0.30), 0, 1) × 100
S_RPM = clamp((RPM − Base_RPM×1.25) / (Base_RPM×0.35), 0, 1) × 100
S_GSR = clamp((Base_GSR×0.80 − GSR) / (Base_GSR×0.30), 0, 1) × 100
Stress = S_HR×0.4 + S_RPM×0.4 + S_GSR×0.2
HR 起點 1.20× 基準、頂點 1.50× 基準 · RPM 起點 1.25×、頂點 1.60× · GSR 訊號跌破 0.80× 才計分、跌至 0.50× 滿分 · 越高 = 越緊張

緊張判定閾值預覽
S_HR = clamp((BPM − Base_BPM×1.20) / (Base_BPM×0.30), 0, 1) × 100  → 0 分在 1.20×、100 分在 1.50×
S_RPM = clamp((RPM − Base_RPM×1.25) / (Base_RPM×0.35), 0, 1) × 100  → 0 分在 1.25×、100 分在 1.60×
S_GSR = clamp((Base_GSR×0.80 − GSR) / (Base_GSR×0.30), 0, 1) × 100  → GSR 須跌破 0.80×Base 才計分、跌至 0.50× 滿分
Stress = S_HR × 0.4 + S_RPM × 0.4 + S_GSR × 0.2
LTI = Peak×0.4 + Avg×0.4 + 事件密度×0.2  （事件密度 = Score ≥ 100 的點數占比 × 100，報告頁顯示）
